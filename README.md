VätternData

Independent analytics dashboard for 2026 Vätternrundan results.

VätternData lets you search riders by bib number or name and explore race
results through split times, ranking evolution, overtakes, finish histograms,
comparison charts, DNF stats, and weather difficulty.

Covered events include Vätternrundan 315 km, Halvvättern, Tjejvättern,
Vätternrundan 100 km, and MTB-Vättern.

The project is inspired by the clear race-analysis experience of vasanerd.se.

This site is not affiliated with Vätternrundan. All data is sourced from
publicly available race results. Race evolution is inferred from timing mats,
not GPS.
